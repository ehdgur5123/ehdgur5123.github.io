# 템플릿 리터럴

**템플릿 리터럴**은 문자열과 변수, 식을 섞어서 하나의 문자열을 만드는 표현 형식입니다.

백틱(```)기호를 사용해 문자열을 묶고, 변수 부분만 `${}` 감싸주면 됩니다.

```javascript
let num1 = 10;
let num2 = 20;

// 연결 연산자를 사용할 경우
console.log()
```
