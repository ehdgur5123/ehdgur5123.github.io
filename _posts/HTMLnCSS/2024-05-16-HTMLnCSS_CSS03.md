## 정렬과 간격

```css
p {
  /* left, center, right, justify */
  text-align: left;

  letter-spacing: 0;
  word-spacing: 0em;
  line-height: 1.5em;
}
```

### text-align

1. left : 왼쪽 정렬
2. center : 중앙 정렬
3. right : 오른쪽 정렬
4. justify : 양쪽 정렬
