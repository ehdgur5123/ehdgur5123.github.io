- *선택자(Selector)* : 꾸밀 요소(들)을 선택

- *선언(Declaration)* : 꾸미기 원하는 요소의 속성을 명시

- *속성(Property)* : 선택자를 꾸밀 수 있는 방법

- *속성 값(Property value)* : 속성의 값
