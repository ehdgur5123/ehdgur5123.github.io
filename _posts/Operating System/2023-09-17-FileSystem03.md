# 저널링 파일 시스템

- **시스템 크래시** : 전원이 나가거나 치명적인 오류로 인해 컴퓨터가 강제로 종료되어 버리는 상황

- **저널링(journaling) 기법** : 작업 로그를 통해 시스템 크래시가 발생했을 때 빠르게 복구하기 위한 방법

    4. 작업 중 시스템 크래시가 발생하면 로그 영역을 읽어 크래시 발생 당시 어떤 작업을 수행 중이었는지 알아낸 다음 해당 작업을 완료
